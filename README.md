# briq-frontend

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Run backend mockup server
```
npm run back-end
```

### Compiles and hot-reloads and run backend mockup server
```
npm run start-app
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
